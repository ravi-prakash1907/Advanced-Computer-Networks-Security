# Advanced Computer Networks Security

The primary significance of this repository is to hold various codes, programs and scripts related to syllabus of _Advanced Computer Networks Security_. Along with that, it would also consist of a plenty of self developed tools related to networking cum cyber security domain.  

[`Submissions`](Submissions/) directory holds all those class projects that are submitted for evaluation.  
[`ns2`](ns2) directory holds the practical implementations of networking concepts using Network Simulator 2.
